# Changelog

All notable changes to this repository are documented in this file.

## [0.1.1] - 2026-03-13

### Changed
- Adjust successful tool background colors to better match the Tomorrow Night Burns palette.
- Use the `themes/` directory manifest entry for pi theme discovery.

## [0.1.0] - 2026-03-13

### Added
- Initial public release of the Ghostty-inspired pi theme pack.
- Added four Tomorrow Night Burns variants:
  - `tomorrow-night-burns-literal`
  - `tomorrow-night-burns-semantic`
  - `tomorrow-night-burns-high-contrast`
  - `tomorrow-night-burns-steel`
